# Project 06-3D RPG

# References

Textures from: https://freepbr.com/
Models and animations from: Mixamo.com
Audio from: Soundtrap.com
Gun model asset from: kenny assets

# Implementation

- User-controlled third-person 3D player with implemented "Remy" model
- Camera follows the movement of the player
- Player model is animated: Mixamo "Firing Rifle" animation
- AI agents "Enemies" that navigate the environment
- Dialogue conversation with an agent who assigns the objective
- Report of the player's score when collecting boxes and destroying enemies
- Probuilder level
- Player has hand held weapon that is used
- Spotlight and Directional lighting
- Materials for the build and boxes
- Background music for the dialogue interaction (Extra)
- End scene when score reaches 15 (Extra)
- End scene aplause audio (Extra)
